# ChatApp
在线聊天系统

## 工具版本
| 工具          | 版本                                      |
| ------------- | ---------------------------------------- |
| node.js (nvm) | 16.13.1                                  |
| yarn          | 1.22.17                                  |
| @vue/cli      | 4.5.15 (requires @vue/cli-init)          |
| mysql8        |                                          |
| redis-3.2     |                                          |

## 启动项目
* `net start mysql`
* `redis-server`
* `yarn start`
